广东省高校一览

发表于 2017-06-16 | 分类于 应届生填报志愿 | | 阅读次数

都说“一冲二稳三保底”，那到底哪些学校可以冲呢？

由于我不清楚17年的分数线具体情况，就弄个16年的给各位参考一下。

【仅供参考，请以指南书为准】

先是16年的分数线：

从案例(选择广东考生广东高校）看填报（第一志愿组）

1.一本文科（2016年），为例560-600分之间，排位在3400—340之间

可以选择香港中文大学（深圳）、中山大学、华南理工大学、暨南大学

2. 文科520-559分之间，排位在15000—3500之间

可以选择深圳大学、华南师范大学、广东外语外贸大学、汕头大学、南方医科大学、华南农业大学、广州大学、北京师范大学—香港浸会大学联合国际学院、广东工业大学、广州中医院大学

3.文科519—480分之间，排位42000—17000之间

（PS：只有达到一本线即可报考广东海洋大学、东莞理工学院、佛山科学技术学院、广东医科大学的一本招生专业，专业基本任选，否则也可以在这个分数段填报）

可以选择广州医科大学、广东财经大学、广东金融学院、广东技术师范学院、广东药科大学、广东第二技术师范学院、仲恺农业工程学院、五邑大学、惠州学院、广州体育学院、广州警官学院、广州美术学院、肇庆学院、广州航海学院、韶关学院、岭南师范学院、嘉应学院、韩山师范学院、广东石油化工学院、北京理工大学珠海学院。

4.文科在479-439之间，排位在80000—42000之间

可以考虑吉林大学珠海学院、广东外语外贸大学南国商学院、华南理工大学广州学院、中山大学南方学院等；第二志愿组可以考虑广东财经大学华商学院北京师范大学珠海分校、广东工业大学华立学院、中山大学新华学院中山大学南方学院、广东财经大学华商学院、电子科技大学中山学院、广东工业大学华立学院、中山大学新华学院、广东培正学院、广州商学院、广州大学松田学院、广州大学华软软件学院等；

第二志愿组可以考虑的院校有广东白云学院、广东东软学院、广东海洋大学寸金学院、华南农业大学珠江学院、广东技术师范学院天河学院、广州工商学院、广东科技学院等。

从案例(选择广东考生广东高校）看填报（理科第一志愿组）

1.一本理科（2016年），560-600分之间，排位在21000-6600

可以选择济南大学、深圳大学、华南师范大学、南方医科大学、汕头大学、广东外语外贸大学。

2.520分—559分之间，排位在50000—24000之间

可以考虑医科大学、暨南大学、广东工业大学、华南农业大学、广州大学、北京师范大学—香港浸会大学联合国际学院、广州中医药大学。

3.480分—519分之间，排位在90000—50000之间的

可以考虑（PS：只有达到一本线即可报考广东海洋大学、东莞理工学院、佛山科学技术学院、广东医科大学的一本招生专业，专业基本任选，否则也可以在这个分数段填报）广东财经大学、广州美术学院、深圳大学、广州要接大学、广东技术师范大学、广东第二师范学院、广东警官学院、仲恺农业工程学院、惠州学院。

4.400分到479分之间，排位在190000—96000之间

可以考虑广东海洋大学、佛山科学技术学院、广东医科大学、肇庆学院、广州航海学院、韶关学院、岭南师范学院、嘉应学院、韩山师范学院、广东石油化工学院、北京理工大学珠海学院。吉林大学珠海学院、广东外语外贸大学南国商学院、华南理工大学广州学院、中山大学南方学院等；第二志愿组可以考虑广东财经大学华商学院北京师范大学珠海分校、广东工业大学华立学院、中山大学新华学院中山大学南方学院、广东财经大学华商学院、电子科技大学中山学院、广东工业大学华立学院、中山大学新华学院、广东培正学院、广州商学院、广州大学松田学院、广州大学华软软件学院等；

第二志愿组可以考虑的院校有广东白云学院、广东东软学院、广东海洋大学寸金学院、华南农业大学珠江学院、广东技术师范学院天河学院、广州工商学院、广东科技学院等。

题外话

广东省如今规划了新的大学教育格局：双一流》高水平大学（含范围性高校）》高水平理工科大学》应用类本科大学》省市共建高校》一流高职类学院 这六类的高校格局，其中双一流跟高水平大学的范围的高校基本都是在综合类研究类的方向发展的，高水平理工科主要是重点突出理工科的优势，其他的高校就是主要承担培养应用型人才为主，服务地方经济。处于一个好的格规划对一个大学的发展非常重要，我看好海大的未来。
